# Generic PR Review

Use para preparar revisão ou descrição de PR.

## Checklist

- Resumir problema.
- Resumir solução.
- Listar arquivos alterados.
- Listar validações.
- Listar riscos.
- Indicar breaking changes, se houver.
- Indicar evidências.

## Saída esperada

- Descrição de PR pronta
