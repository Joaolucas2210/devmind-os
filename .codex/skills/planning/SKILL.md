# Generic Planning

Use após discovery e antes de alterar código.

## Objetivo

Criar plano incremental, seguro e testável.

## Checklist

- Definir passos pequenos.
- Listar arquivos prováveis.
- Definir validações.
- Definir riscos.
- Definir rollback quando aplicável.
- Não alterar código nesta etapa.

## Saída esperada

- Plano
- Arquivos impactados
- Validações
- Riscos
