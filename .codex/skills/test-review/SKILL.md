# Generic Test Review

Use após implementação.

## Checklist

- Rodar testes disponíveis.
- Rodar lint quando disponível.
- Rodar build quando disponível.
- Rodar typecheck quando disponível.
- Rodar docker compose config quando aplicável.
- Revisar diff.
- Reportar validações que não puderam ser executadas.

## Saída esperada

- Comandos executados
- Resultado
- Problemas encontrados
- Correções feitas
