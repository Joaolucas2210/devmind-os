# Generic Security Review

Use para revisar segurança e privacidade.

## Checklist

- Verificar secrets.
- Verificar tokens.
- Verificar `.env` real.
- Verificar PII em logs, URLs e erros.
- Verificar permissões/autorização.
- Verificar input validation.
- Verificar comandos destrutivos.

## Saída esperada

- Riscos encontrados
- Mitigações
- Pendências
